## Backend


#### Backend part with spring boot

> Start

```bash
# start the server
$ mvn spring-boot:run
```



The Backend is start in [http://localhost:8080](http://localhost:8080) 

Start frontend and go to [http://localhost:8081](http://localhost:8081) in your browser to .


> Used technologies

- Spring boot
- JPA Hibernate
- Spring Security
- JWT
- Swagger
- Mongobee
- REST
- Embeded MongoDB
